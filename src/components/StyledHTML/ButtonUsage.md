# Button

This replaces the default HTML button with a styled button that contains the following styles:

- PrimaryLight
- PrimaryMedium
- Primary
- SuccessLight
- SuccessMedium
- Success
- InfoLight
- InfoMedium
- Info
- WarningLight
- WarningMedium
- Warning
- DangerLight
- DangerMedium
- Danger


And draws themes from DesignSystem/theme.js


## Usage
```html
<n-button :flavor="Primary"></n-button>>
```
